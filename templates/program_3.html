Change Alphabetic Cases in a String

To change the cases of the alphabetic characters in the string and print it
If 'a' change it to 'A', If 'B' change it to 'b'
Input Format
Input represents a string with only alphabetic characters
Output Format
Print the changed alphabetic cases of the string
Constraints
Length of the string
1<= length_of_string <=100

Sample Input 1:
cOdinGbUzz
Sample Output 1:
CoDINgBuZZ

Sample Input 2:
helloWORLD
Sample Output 2:
HELLOworld
